VOIDRUNNER
A data-driven, deterministic twin-stick roguelite. This whole demo — the physics, the RNG, the collision, the replay verifier — is a pure-JavaScript port of the game's engine-agnostic core, reproducing the C# simulation bit-for-bit. No install, no engine, no server.
